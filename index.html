Projetos

Dashboard de Performance

Desenvolvido durante o bootcamp de Google Data Studio

O objetivo do projeto foi desenvolver um painel para o acompanhamento da performance geral de um negócio na área da educação, que possui canais de aquisição pagos (Google e Facebook). O painel será usado pela gerência geral e de marketing.

Ferramenta: Google Data Studio

Ecommer Analysis

Painel desenvolvido durante o curso CDO - Convertendo Dados em Oportunidades

O projeto teve como objetivo responder as seguintes perguntas: quantidades de produtos vendidos; distribuição de vendas por método de pagamento; total de receita, ticket médio; top 10 produtos mais vendidos; porcentagem de produtos em atraso.

Ferramenta: Google Data Studio

Maven Airlines

Desenvolvido para o desafio promovido pela Maven Analytics

O objetivo do projeto foi analisar uma base de pesquisa de satisfação de uma empresa aérea, com mais de 120.000 registros que incluem informações sobre voô, tipo de viagem, limpeza, conforto e experência em geral.

Ferramenta: Power BI

Sales Report

Desenvolvido a partir da base de dados Microsoft Contoso BI Demo Dataset for Retail Industry. A análise exploratória foi realizada utilizando o SQL Server, o dashboard e o relacionamento entre as tabelas foram desenvolvidos no Power BI. Para a análise foram considerados dados de 2008 e 2009, e as principais informações relacionadas as vendas, lojas, produtos e suas categorias.

Ferramentas: SQL Server e Power BI

YouTube Analysis

Desenvolvido para o desafio promovido pela Enterpise DNA

Desenvolvido a partir do dados do YouTube da Enterpise DNA, O objetivo do projeto foi realizar uma análise exploratória para enternder o crecimento do canal e o perfil da sua audiência.

Ferramenta: Power BI
